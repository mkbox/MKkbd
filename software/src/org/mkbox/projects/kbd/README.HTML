Вступление

Данная программа предназначена для изменения раскладки контроллера кнопочных панелей. Контроллер поддерживает 4 раскладки, выбор активной производится путём установки перемычек J4:
  • 1 раскладка - нет перемычек
  • 2 раскладка - установлена первая перемычка
  • 3 раскладка - установлена вторая перемычка
  • 4 раскладка - установлены обе перемычки
Актуальную версию программы можно скачать на странице проекта - MKBox Workshop

Требования

Программа поставляется в двух вариантах - только приложение, требующее установленного Java Runtime Environment (JRE) версии 1.8 и в виде пакета, содержащего все необходимые библиотеки для запуска. Второй вариант занимает значительно больше места на жёстком диске (при распаковке около 200Мб), чем первый (около 3Мб), но не требует дополнительного ПО.

Зависимости

В программе используются 2 сторонние библиотеки (включены в пакет и не требуют установки):
  • JNA - библиотека для простого доступа к нативным общим библиотекам без использования JNI Java Native Access (лицензии LGPL 2.1 и Apache License 2.0)
  • PureJavaHidApi - библиотека для кроссплатформенного доступа к HID устройствам, не требует установки фильтров для USB устройств и установки дополнительных приложений. Страница проекта Pure Java HID-API (лицензия BSD)

Установка

Программа не требует установки и работает "из коробки".

Настройка/Использование

Различные версии контроллеров используют разное количество клавиш, для того, чтобы получить количетсво используемых клавиш, нужно нажать кнопку Connect для получения настроек от контроллера.
Для начала редактирования раскладок необходимо сделать следующие действия:
  • нажмите клавишу Edit layouts
  • сделайте двойной клик на ячейке, которую хотите изменить
  • после открытия окна нажмите клавишу на клавиатуре, которую хотите установить
Чтобы не изменять раскладки при каждом запуске программы, рекомендуется их сохранить, нажав кнопку Save (чтобы открыть сохранённый ранее файл, нажмите кнопку Load. Для записи раскладок в контроллер необходимо выполнить следующее:
  • нажать кнопку Connect.
  • нажать кнопку Write.
Чтобы считать текущие настройки из контроллера:
  • нажать кнопку Connect.
  • нажать кнопку Read.
После нажатия кнопки Connect в случае успешного подключения к контроллеру изменится на кнопку Disonnect и станут активными кнопки Read и Write. При отключении контроллера или нажатии кнопки Disconnect кнопки Read и Write станут неактивными и появится кнопка Connect. Поддерживаемые клавиши отмечены зелёным
